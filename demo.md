Q: Is my data safe?
A: Yes! All data is stored in Firebase with user authentication. Only you can see your data.
Q: Can I use this offline?
A: Partially. The app needs internet for first load, but has some offline capability.
Q: Can multiple people use the same account?
A: No, each email is one account. For family use, each person should have their own account.
Q: Can I delete my account?
A: Currently no, but you can stop using it. Data will remain in Firebase.
Q: How do I backup my data?
A: Use Export & Backup feature to download CSV/Excel files regularly.
Q: What if I make a mistake?
A: Tap the transaction to edit or delete it. Changes sync immediately.
Q: Can I track different currencies?
A: Currently only â‚¹ (Rupees). Multi-currency coming in v1.2.0!
Q: Does it work on mobile?
A: Yes! Open in mobile browser. Native app (APK) coming soon!
Q: How do transfers work?
A: Transfers move money between YOUR accounts. Not for sending to others.

A: Share the app link! Each friend creates their own account.
